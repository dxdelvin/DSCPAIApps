    Quick Start

    How to Use This Builder

    1. 1
      Upload files and assign roles Choose which files should guide AI, which should be attached to the page, and which images should appear in the content body.
    2. 2
      Generate and review the draft Edit the AI summary, adjust the title, and tune the Confluence storage format directly inside the workspace.
    3. 3
      Refine or regenerate Ask AI for another version if the structure is off, or keep polishing the storage format manually until it is ready.
    4. 4
      Publish with your PAT Provide Confluence URL, Space Key, Parent Page ID, and PAT, then publish the page and attachments in one step.

    Business Tips

    • Use PDFs with readable text for the strongest first draft.
    • Keep only the most relevant screenshots as display images so the page stays clean and purposeful.
    • Use the storage format warning panel to catch missing attachment references before posting.
