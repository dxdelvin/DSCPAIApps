Diagram Type Guide

Each type below shows a visual preview and when to use it. Select the type that best fits your content.

🔀 Flowchart

Sequential steps, decisions, and process loops. Great for workflows and procedures.

🧠 Mind Map

Radial brainstorming around a central concept. Great for topics with many related subtopics.

🔄 Sequence Diagram

Step-by-step message flow between actors or systems over time.

🗃️ ER Diagram

Database tables with attributes and relationships. Ideal for data models and schemas.

🏢 Org Chart

Hierarchical reporting structure. Great for teams, departments, and corporate structures.

🏊 Swimlane

Cross-functional process with lanes per role/department. Shows who does what across teams.

🧱 Block Diagram

High-level system components and their connections. Great for architecture overviews.

📦 Class Diagram

OOP class structures with attributes, methods, and inheritance. For software design.

⚡ State Diagram

States and transitions within a system lifecycle. For modelling behaviour over time.

🌐 Network / Architecture

Network topology or system architecture with nodes and connections.

📅 Timeline

Chronological events, milestones, or project phases on a linear axis.

🌲 Tree / Hierarchy

Hierarchical breakdown structure like taxonomy, WBS, or decision trees.

📈 Data Flow

How data moves through processes and storage. Great for ETL pipelines and data architectures.

👤 Use Case

Actor interactions with system use cases. Great for requirements and user stories.

🔧 Component Diagram

Software components with provided/required interfaces. For software architecture design.

🚀 Deployment

Physical deployment of software on servers/nodes. For infrastructure and DevOps planning.

🎯 Activity Diagram

Detailed process flow with forks, joins, and decisions. More detailed than a flowchart.

⭕ Venn Diagram

Overlapping sets showing shared and unique elements. Great for comparisons and relationships.

📊 Gantt Chart

Project tasks plotted over time. Ideal for schedules, project planning, and timelines with durations.

🖼️ Wireframe

Low-fidelity UI layout sketch. Great for app/website design planning and UX documentation.
