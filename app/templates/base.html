What's New Updates?

Mar 24th, 2026: NEW v2.6.0

  • Updated the Logo, Colors, Font according to the BSH Design Guidelines across the application.
  • AI Diagram Generator now includes a visual diagram guide with previews to help users choose the right diagram type faster.
  • Added clear step-by-step guides in PowerPoint Creator, AI Diagram Generator, and Audit Checker for easier first-time use.
  • PowerPoint Creator and AI Diagram Generator now support PDF, PNG, JPG, and JPEG uploads.
  • PowerPoint Creator App now includes 4 new SmartArt layouts, Improved the Presentation output, added a orange theme button
  • Added Easter Egg Button in navigation for a festive theme. (Testing Feature)

  • Mobile User Experience significantly improved with optimized responsive CSS across all tools.
  • Added a new "Upload & Analyze" feature in the Signavio BPMN Builder, complementing the existing Form Builder mode.
  • Launched AI Diagram Generator to analyze uploaded PDFs and generate downloadable draw.io diagram types with refined result cards.

  • Learn Signavio app linked - Interactive learning website with games to master BPMN modeling.
  • Functional Spec content and structure updated for better document quality.
